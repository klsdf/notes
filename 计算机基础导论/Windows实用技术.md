# 常见报错

## 禁止运行脚本

1. 首先打开power shell
2. 输入下面的代码

```powershell
Start-Process powershell -Verb runAs
```

3. 确认
4. 输入下面的代码

```powershell
set-ExecutionPolicy RemoteSigned
```

5. 按Y或A
6. 重启
